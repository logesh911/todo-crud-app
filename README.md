# 📝 To-Do CRUD App

A full-stack To-Do application built using:

- Node.js + Express
- HTML, CSS, JavaScript
- File-based storage (JSON)

## 🚀 Features
- Add tasks
- Edit tasks
- Delete tasks
- Mark as completed
- Dark mode 🌙
- LocalStorage backup 💾

## ▶️ Run Locally

```bash
npm install
node server.js
